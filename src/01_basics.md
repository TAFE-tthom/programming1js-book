# CLI, Basics and Tools

We'll cover the following:

* Command Line Interface/Shell

* Program and Process

* Build Tools

* Shell basics and Running your programs.


After installing and getting setup with your code, you will need to get familiar with the command line. The command line will be a fairly standard interface that you will use to interact with programs you have made and execute your programs.

It is also one in which, when understanding the foundations here, will enable you to compose many applications together.
